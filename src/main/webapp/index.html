Sourced?

Implicitly sourced is a free host for sxr processed scala source files. The sxr document publishing protocol is currently implemented by the sbt plugin sxr-publish. Publishing documents requires a secret key associated with your project's orgId (com.project.org). To obtain a private key for your project, please contact @n8han or @softprops.

Be Sourced

> cd path/to/project

> cat project/plugins/Plugins.scala
import sbt._
class Plugins(info: ProjectInfo) extends PluginDefinition(info) {
  val sxr_publish = "net.databinder" % "sxr-publish" % "0.1.8-SNAPSHOT"
}

> cat project/project/Project.scala
import sbt._
class Project(info: ProjectInfo) extends DefaultProject(info) with sxr.Publish

> cat ~/.sourced.implicit.ly
orgid=orgs3cr3t

> sbt

> preview-sxr

> publish-sxr
